# ultima_messis
Um projeto pessoal de jogo na Unity: Você é um prisioneiro otimizando uma horta na varanda da sua cela. Gaste energia para plantar e colher. Descubra combos de plantas baseados em suas Tags ([Raiz], [Fruta]). Bata a cota semanal ditada pelo "Jornal" (que também modifica as regras!). Falhe, receba seu "Relatório" e compre upgrades permanentes.
